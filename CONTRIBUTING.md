# Contribuindo para o Ebook de Lógica de Programação

Obrigado pelo interesse em contribuir!

## Como contribuir

Você pode ajudar de várias formas:

* Corrigindo erros de ortografia.
* Corrigindo erros de programação.
* Melhorando exemplos de código.
* Sugerindo novos exercícios.
* Indicando referências confiáveis.
* Melhorando explicações existentes.

## Regras

* Utilize linguagem clara e objetiva.
* Mantenha a formatação Markdown do projeto.
* Sempre cite fontes confiáveis quando adicionar conteúdo técnico.
* Evite conteúdo copiado de terceiros.

## Processo

1. Faça um Fork do projeto.
2. Crie uma branch para sua alteração.
3. Realize as modificações.
4. Abra um Pull Request descrevendo as mudanças.

## Objetivo

Criar um material gratuito e completo para estudantes de programação.
